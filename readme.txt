== Description ==

Info Box summarize your Continent in gird box layout. With help of this plugin you can display your information in a box shape with unlimited color scheme and options.

== Installation ==

1. Upload the entire `info-box` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Info Box menu to customize this plugin.
4. Copy your publish Info Box shortcode and then paste into your page or post to display it.
